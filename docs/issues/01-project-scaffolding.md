# 01 - Project scaffolding + GitHub Pages deploy

## What to build

Initialize the project with Vite + React + TypeScript + shadcn/ui. Set up a GitHub Actions workflow that builds the app and deploys it to GitHub Pages on push to `main`. The result is an empty app (placeholder landing page) live at GitHub Pages.

## Acceptance criteria

- [ ] Vite + React + TypeScript project initialized with shadcn/ui configured
- [ ] GitHub Actions workflow builds and deploys to GitHub Pages on push to `main`
- [ ] App is accessible at the GitHub Pages URL
- [ ] Mobile viewport meta tag is set
- [ ] Landing page renders with placeholder content

## Blocked by

None - can start immediately
